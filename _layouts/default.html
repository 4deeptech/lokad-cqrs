Lokad.CQRS is a .NET framework and a set of guidance materials for building distributed and scalable applications to be run on-premises or in the cloud. This project helps to design and develop decoupled systems locally and bring them to the distributed environments later. This is draft of Lokad.CQRSv3 guidance, scheduled for release in fall 2011. In the meantime, v2 can be downloaded as PDF.
